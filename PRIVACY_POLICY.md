# Privacy Policy

Last updated: June 2026

## Overview

KeyMaker is a browser extension designed to generate deterministic passwords locally on the user's device.

## Data Collection

KeyMaker does not collect, store, transmit, sell, or share any personal information.

The extension does not:

* Create user accounts
* Store passwords on external servers
* Track browsing activity
* Use analytics services
* Use advertising services
* Collect personally identifiable information

## Local Processing

All password generation is performed locally within the user's browser.

The information entered by the user never leaves the device.

## Third-Party Services

KeyMaker does not send user data to any third-party service.

## Open Source

The source code of KeyMaker is publicly available on GitHub and can be reviewed by anyone.
